Aula -4 Criando a pasta do projeto
	no terminal 'CMD' digite mkdir /c/NinjaDoCypress
	acessar o site gitHub.com/ninjadocypress/webdojo e fazer o 'FORK' do projeto
	escolher depois a opção 'Code' e deixar o SSH marcada e copia o codigo
	dentro da pasta usando o terminal clona o projeto
	Vai precisar de permissão para clonar o projeto,
	ai digite no terminal: git config --global user.name "Teófilo Messias Filho" + ENTER
	Depois digite: git config --global user.email "teofilo.messias@gmail.com" + ENTER
 	e digite: ssh-keygen -t rsa -b 4096 + ENTER
	senha é: key
	pegar o caminho aonde esta escrito ' Your identification has been saved in /c/Users/jailz/.ssh/id_rsa.pub'
	ai ainda no terminal digite: cat + o caminho /c/Users/jailz/.ssh/id_rsa.pub
	ficando assim: cat /c/Users/jailz/.ssh/id_rsa
	copia o código gerado e vai no GitHub clica no usuário e vai na opção [SETTINGS] vai na opção SSH and GPG Keys
	entra no botão [New SSH key]
	volta no CMD e digite git clone e novamente passe a URL do site:
	git clone gitHub.com/ninjadocypress/webdojo + ENTER
